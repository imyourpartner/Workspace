# MySQL

### Seleccionar una base de datos.
```mysql
USE nombreBaseDeDatos;
```

### Retorna cuál es la base de datos seleccionada actualmente.
```sql
SELECT DATABASE();
```
### Mostrar todas las tablas que contenga una BD seleccionada
```mysql
SHOW TABLES;
```
### Muestra la estructura como está compuesta una tabla
```mysql
SHOW COLUMNS FROM  nombreDeLaTabla;
```
